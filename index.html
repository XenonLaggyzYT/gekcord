Gekcord

A subcommunity from Discord in which you can add bots, advertise servers, and an messaging system without
needing to join any kind of discord server.

Learn More

Gekcords Hopes & Dreams

Gekcord wants to help raise popularity of servers and bots on Discord
while providing a nice commmunity and a great experience we want to develop
features like:
  • A system to advertise servers
  • A system to advertise your bots
  • Profiles
  • Minigames
  • Intergrated messaging system without needing to join a server

Social Development

After we finish basic front end development we will start working with Discord's API
to allow users to sign in with Discord as their account on Gekcord. Messaging systems
however is not our priority, we want to finish server and bot advertising first.
